## Repeatable Character Style Description

**Angular satirical software-engineer character art** with a **Gravity Falls-inspired cartoon vibe**, but pushed into a more **polygonal, faceted, low-poly 2D look**. Characters should feel like funny tabletop game archetypes: confident, slightly smug, nerdy, overloaded with tech gear, and full of programmer in-jokes.

The key is: **almost every shape should be built from straight lines.**

Faces, hats, ears, hair, clothes, shoes, bags, and hands should feel **sharp, blocky, and geometric**, not soft or rounded. The sample works because the hat is almost a collection of trapezoids, the ear is diamond-like, the jaw and cheeks are made from clean straight cuts, and the clothes have hard angular folds.

## Core Visual Rules

Use:

**Shape language**

* Polygonal / faceted construction
* Straight-line edges
* Trapezoid hats
* Diamond-shaped ears
* Sharp jawlines
* Angular cheek planes
* Pointed noses
* Blocky shoes
* Geometric hands and fingers
* Clothing folds made of triangles and hard creases

**Linework**

* Thick black outlines
* Clean cartoon inking
* Strong silhouette
* Minimal tiny detail unless it supports the joke

**Shading**

* Flat cel shading
* Faceted shadow shapes
* No painterly gradients
* Shadows should look like polygon pieces, especially on pants, hoodies, bags, and faces

**Character personality**

* Half-lidded eyes
* Smug grin or sarcastic expression
* Slightly awkward but cool posture
* “I know this system is broken and I still deployed it” attitude

**Software-engineer props**

* Laptop with fake stickers
* Coffee mug
* Energy drink
* Mechanical keyboard
* Headphones
* Badge lanyard
* Backpack full of gadgets
* Rubber duck
* Game controller
* Sticky notes with short jokes
* Code symbols like `</>`, `{}`, `>_`, `404`, `BUGS`, `SHIP`, `DEBUG`

## Reusable Prompt Template

You could use this as your base prompt every time:

> Create a full-body original character illustration for a satirical tabletop game about software engineers. Use an angular, polygonal, Gravity Falls-inspired cartoon style. The character should be built from clear straight lines and faceted shapes: trapezoid hat shapes, diamond-like ears, sharp jaw and cheek planes, pointed nose, angular hair chunks, blocky hands, geometric shoes, and clothing folds made from triangles and hard creases. Avoid soft rounded forms or bubbly proportions. Use thick black outlines, flat cel shading, faceted shadows, a clean off-white background, and a subtle polygonal ground shadow.
>
> The character should feel funny, confident, nerdy, and slightly smug, like a software-engineer archetype for a tabletop card game. Add tech accessories such as laptop stickers, coffee, keyboard, headphones, bug charms, sticky notes, code symbols, and satirical programmer details. No card frame, no stats, no title, just the centered full-body character.

Then add the specific character:

> Accessories: [bag, laptop, props, jokes].
