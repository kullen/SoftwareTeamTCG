# SoftwareTeamTCG

software team tcg is an almost parody game of being a software engineer.  Players, software engineers, are on a mission to complete development of their very cool app.  But there always seems to be something working against them too, scope creep, server issues, changes in requirements, burnout (all light hearted and amusing)

Characters are software engineers that have varying levels of experise in UI, Backend, and Database development. There are juggernaut senior engineers that are great at all things and speciallized engineers that may only be great at 1 thing. and junior engineers that aren't really good at anything, but they are more energetic than senior engineers.
